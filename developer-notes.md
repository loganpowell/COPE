## Global (CLI) Dependencies

These are required to be installed globally for `scripts`:

- `npm-check-updates`
- `jsdoc`
- `concurrently`
- `madge`
- `typescript`
- `jest`
- `parcel`
- `gh-pages`

## Last Compilation Environment

- Node Version: 12.20.1/x64
- OS: Windows 10

# Step by Step Guide

1. `amplify init`
2. `amplify add api`
3. `amplify add auth`
